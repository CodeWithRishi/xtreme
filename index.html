Command List

Help Commands

Usage Description
_help commands To get help for other helpfull commands
_help info To get help for information commands
_help memes To get help for meme commands
_help games To get help for game commands
_help search To get help for search commands
_help pokemon To get help for pokemon commands
_help admin To get help for admin commands
_help fun To get help for fun commands
_help aki To get help for character guesser(akinator) commands

Information commands

Usage Description
_serverinfo Get information about the server
_count members To get help for information commands
_avatar To get or see your avatar
_avatar (member id or mention) Get your information.
_userinfo (mention or member id) Get information about mentioned user.

Meme commands

Usage Description

Game commands

Usage Description
_tictac (mention) Challenge your opponent for a game of tic tac toe .
_profile To get your profile data in tic-tac-toe.
_lb To get the global leaderboard for tic-tac-toe .

Music commands

Usage Description
_play (song name) Plays the song which you passed .
_pause Pause the current song.
_resume Resume the song which was paused .
_stop Stops the current song .

Search commands

Usage Description
_yt search (query) Search a youtube video . Included with - a youtube video.
_yt data (query) Get the data youtube video you search like view , channel name
_go search (query) Search on google . *Included with - a screenshot , a link browser
_wiki search (query) Gives results from wikipedia..

Pokemon commands

Usage Description
_pokemon (pokémon_name) Get pokémon data from pokédex
_show (pokémon_name) Get pokémon image

Admin commands

Usage Description
_welcome (welcome_channel) (welcome message) To set or update the settings for welcome channel and welcome message EXAMPLE:- _welcome (mention channel) (welcome message with #member to mention the member who joins this server)
_left (leave_channel) (leave message) To set or update the settings for leave channel and leave message EXAMPLE:- _leave (mention channel) (leave message with #member to mention the member who leaves this server)
_log (leave_channel) To set or update the channel for audit logs
